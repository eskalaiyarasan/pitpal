# pitpal
PitPal is a configurable, schema-driven rule engine and tooling framework for pit-based traditional board games (like Pallanguzhi variants), designed to support:  
     Multiple rule variants  
     Runtime configuration  
     Localization  Automation (CI/CD)  
     Structured validation using JSON Schema (Draft 2020-12) 

It is not just a game. . It is a Development Program.
It enhances mathematical skills, concentration, and fine motor skills
    -> Mathematical Skills: Improves quick mental calculation, addition, and subtraction as players count and distribute seeds.
    -> Cognitive Development: Boosts memory, strategic planning, and observation skills.
    -> Motor Skills & Coordination: Enhances fine motor skills and hand-eye coordination by handling small, intricate seeds or shells.
    -> Focus and Patience: Develops, concentration and teaches patience and emotional resilience in handling win-loss outcomes.
    -> Social & Cultural Bonding: Fosters interaction and social bonding, traditionally played during family gatherings and festivals.
